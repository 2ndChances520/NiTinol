Welcome to **Bound by Design**. This project is a beautifully formatted, web-based speech script designed to be read seamlessly from a tablet or printed perfectly for a podium. It tells the incredible true story of Nitinol (a shape-memory alloy) and uses its unique physical properties as a profound metaphor for the human condition, the fall, and our ultimate restoration through Christ.

---

## 🌟 The Core Message

This script takes the audience on a journey through:
1. **The Laws of Creation:** How the four fundamental forces of the universe (like the Strong and Weak forces) mirror God's sustaining power and the reality of spiritual decay.
2. **The Discovery of Nitinol:** The fascinating 1958 discovery at the Naval Ordnance Laboratory of a metal that "remembers" its shape.
3. **The Refiner's Fire:** A brilliant analogy demonstrating how the Shape Memory Effect parallels Genesis 1:26. Life twists and bends us, but when the "Refiner's Fire" is applied, we are compelled by God's spiritual law to snap back into our original design: *The image of Christ.*

---

## ✨ Features

This isn't just a standard text document; it is a custom-coded HTML/CSS reading environment built specifically for public speakers and pastors.

* **🌙 Teleprompter Mode (Dark UI):** Features a sleek dark mode (`#0d1117`) with gold accents to reduce eye strain when speaking from a glowing tablet or laptop on stage.
* **🖨️ Podium-Ready Print Styles:** Built-in `@media print` CSS automatically strips away the dark background, removes shadows, and switches to a high-contrast black-and-white layout the moment you press `Ctrl+P` or `Cmd+P`.
* **🏷️ Visual Cues:** Utilizes color-coded tags (`Concept`, `Scripture`, `Narrative`, `Side Note`, `Delivery`) so the speaker instantly knows the context and tone of the next paragraph at a glance.
* **🖋️ Elegant Typography:** Uses Google Fonts (*Playfair Display*, *Cinzel*, and *Lato*) to create a manuscript that feels as profound as the words written on it.

## 🏃‍♂️ How to Use

Because this script is entirely self-contained within a single HTML file, there is zero setup required:

1. **Download:** Clone or download this repository.
2. **Read Digitally:** Double-click the `index.html` file to open it in any web browser. Prop your tablet or laptop on the podium and read away!
3. **Print Hardcopy:** While the file is open in your browser, press `Ctrl+P` (Windows) or `Cmd+P` (Mac). You will see the dark mode automatically vanish, leaving you with a perfectly formatted, ink-saving printable document.
